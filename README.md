# CraftTrend Radar｜手工趋势雷达（小红书手工趋势运营作品）

一个面向“手工趋势运营”的轻量工具：聚合 **国内/海外最新手工趋势**，支持 **关键词检索、标签过滤、收藏、导出、生成小红书选题模板**，用于日常选题、周报与内容策划。

> 作品定位：应聘 **小红书｜手工趋势运营** 的作品展示，强调“趋势采集 → 结构化整理 → 内容转化 → 可复用输出”。

---

## 在线体验
- GitHub Pages：`https://<你的用户名>.github.io/<你的仓库名>/`

---

## 主要功能
- **趋势流（国内/海外）**：按时间展示趋势条目（标题/来源/发布时间/标签）
- **检索与筛选**：
  - 关键词搜索（中英均可）
  - 地区筛选：国内 / 海外
  - 时间范围：近7天 / 近30天 / 近半年 / 全部
  - 标签多选过滤
- **收藏夹**：一键 ⭐ 收藏，保存在浏览器（localStorage）
- **一键生成小红书选题**：点击趋势条目自动输出：
  - 3个可用标题
  - 笔记结构（开头钩子/材料清单/步骤/避坑/拍摄建议/互动引导）
  - 推荐话题标签（#手工 #DIY教程 等）
- **导出**：
  - 导出 JSON（方便二次加工）
  - 导出 CSV（方便周报/表格整理）
  - 生成“今日简报”（Top 标签 + 重点趋势列表）

---

## 数据来源与更新机制（可扩展）
当前版本读取仓库内的 `trends.json` 作为数据源，便于稳定展示与快速迭代。

**可选增强（建议开启）**：
- 使用 **GitHub Actions** 定时抓取公开趋势信号（RSS/公开趋势报告入口）并自动更新 `trends.json`
- 推荐信号源：
  - Google News RSS（按关键词聚合国内/海外手工相关新闻/资讯）
  - Pinterest Predicts（年度趋势方向参考）
  - Craft Industry Alliance（行业趋势内容）
  - Etsy Seller Handbook（手作电商趋势入口）

> 注意：纯前端网页直接跨站抓取会遇到 CORS 限制，因此采用“Actions 定时生成 trends.json → 前端读取”的方式更稳定。
